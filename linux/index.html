Linux

Linux is a family of open-source Unix-like operating systems based on the open-source Linux kernel. It has all the features you would expect in a modern fully-fledged Unix, including true multitasking, virtual memory, shared libraries, demand loading, shared copy-on-write executables, proper memory management, and multistack networking including IPv4 and IPv6. In addition it's completely free and anybody can use it.
