Why use Linux

An operating system is software that manages all of the hardware resources associated with your desktop or laptop. Just like Windows, iOS, and Mac OS, Linux is an operating system. So the question is: why should I use Linux?

Reasons why use Linux

According to r/linuxmasterrace some reasons are:

  • Less malware than OSX and Windows.
  • It's faster and more efficient than OSX and Windows.
  • No bloatware included.
  • You can make your desktop look however you want. Literally.
  • You can update all your computer's software with a single click.
  • Runs on pretty much anything, no matter how old it is.
  • You can see every line of code it uses, so you know there's no NSA backdoor. They tried (and failed) to get a backdoor inserted into Linux. There are also no keyloggers built-in, unlike in Windows
  • It costs exactly $0.
  • According to Gabe Newell, it's the future of PC gaming (as is SteamOS).

Is Linux for everyone?

Yes and no. Using Linux is not as scary as many people make it out to be. In some distributions you don't even need to use a terminal, although it is very helpful to use it. However, if you want to use this system efficiently, you need to learn the basic commands and how the directory structure works on this system. So if you have a hard time learning new things related to computers this system is definitely not for you.

Whether linux is for you or not

You should give Linux a chance if you:

  • Care about your own privacy.
  • Want lightweight, fully functional os.
  • Want your system to be YOURS and you can do whatever you want in it.
  • Don't want to have unnecessary things on your computer that you won't use anyway.
  • Want to have full control of your computer.
  • Want to have everything for free.
  • Want to give a "new life" to old computer.

Linux is not for you if you:

  • Want a full support for non open-source, not designed for Linux programs (Like Adobe Photoshop, Visual Studio, etc.).
  • Have a hard time learning things related to computers.
  • Want everything to just work (even if it works out of your control).
