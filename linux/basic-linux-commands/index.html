Basic Linux Commands

To use linux efficiently, you need to know the basic commands

Help commands

help - shows all the commands in system

man <command> - shows manual for the command

<command> --help - shows usage of the command

Commands related to files and directories

ls - shows the contents of the directory

dir - alternative command showing directory contents (from DOS)

pwd - shows the path to the current directory

cd <location> - go to the indicated directory

mkdir <directory_name> - create directory

rmdir <directory_name> - remove directory

mvdir <directory_name> <new_location> - move or rename directory

touch <file_name> - create empty file

cat <file> - display file content

rm <file> - remove file

cp <file> <location> - copy file

mv <file> <new_location> - move or rename file

chmod <permissions> <file/directory> - changing file/directory permissions

chown <user> <file/directory> - changing file/directory owner

chgrp <user> <file/directory> - changing file/directory group

Commands related to system

sudo <command> - use root privileges to execute the command

who - displaying currently logged in users

whoami - displays the name of the user who is logged in to the current session

shutdown - shuts down the system

logout - logs out the user

useradd <options> <user> - adds a user

groupadd <options> <group> - adds a group

passwd <options> <user> - password change

su <user> - change of user

Commands related to processes

ps - shows currently running processes (in current terminal)

kill - kills processes
