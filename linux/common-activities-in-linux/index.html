Common activities in Linux

Some activities in linux, such as updating, are performed frequently and repetitively

System update

Debian and similar systems (Ubuntu, Linux Mint, Pop!_OS, etc.):

sudo apt update && sudo apt upgrade

Red Hat and similar systems (Fedora, CentOS, ClearOS, etc.):

sudo dnf upgrade

Arch Linux and similar systems (Manjaro, Garuda Linux, ArcoLinux, etc.):

sudo pacman -Syu

Slackware and similar systems:

sudo slackpkg update gpg && sudo slackpkg update

Gentoo and similar systems:

sudo emerge --sync && sudo emerge --update --deep --with-bdeps=y @world

Installing software

Debian and similar systems:

sudo apt install <package_name>

Red Hat and similar systems:

sudo yum install <package_name>

Arch Linux and similar systems:

sudo pacman -S <package_name>

Slackware and similar systems:

sudo slackpkg install <package_name>

Gentoo and similar systems:

sudo emerge -av <package_name>

Adding user

sudo useradd -m <user> -s /bin/bash - create a user and set /bin/bash to its default shell

sudo passwd <user> - giving a password to the user

sudo usermod -a -G <group> <user> - adding user to group
